Control Neato
Control your Neato Robot from the web!

This tool allows you to control your Neato Robot from the web, such as on a desktop computer. I created this tool because I was disappointed to find that there was no way to control your Neato Robot from the web, only via apps on your phone and I wanted to be able to control it from my laptop.

To use the tool, simply login below using Neato's oAuth system. This is entirely safe, we do not get access to your account or any credentials. All of this app is open source on Github.
